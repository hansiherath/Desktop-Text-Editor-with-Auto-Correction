# 📝 Desktop Text Editor with Auto-Correction

## 📖 Overview

This project is a desktop-based text editor developed using C++ and the Qt Framework. The application provides essential text editing capabilities along with automatic text correction and sentence capitalization features to enhance the user writing experience.

The project demonstrates GUI application development, file handling, event-driven programming, and text processing using Qt.

---

## 🎯 Objectives

- Develop a user-friendly desktop text editor.
- Implement basic file management operations.
- Provide text editing functionalities.
- Automatically capitalize sentences.
- Demonstrate GUI development using Qt Framework.

---

## ✨ Features

### File Operations

- Create New Document
- Open Existing Files
- Save Documents
- Save As Functionality

### Editing Features

- Cut
- Copy
- Paste
- Undo
- Redo

### Auto-Correction Features

- Automatic sentence capitalization
- Text correction processing
- Detection of punctuation-related formatting issues

### Additional Features

- About Dialog
- Simple and clean graphical interface

---

## 🖥️ User Interface

The application provides a desktop GUI built using Qt Widgets.

Users can:

- Create and edit text documents
- Save notes and text files
- Perform standard editing operations
- Benefit from automatic text formatting assistance
